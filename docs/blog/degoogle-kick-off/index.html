De-Google my life

2024-02-02

Is it too late to set New Year's resolutions? I don't believe so! My new goal for 2024 is to #DeGoogle my life. I've replaced Twitter with Mastodon and have ceased using Reddit in 2023, so this sounds like a logical continuation.

My 18-year-old self would be proud of me. Back in 1999, I abandoned Microsoft and Windows simply out of principle;-) I may not be as hardcore anymore and will make some exceptions at the end, but let's see where this journey takes me...
